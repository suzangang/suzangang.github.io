# 结构赋值

在ES6中,可以使用解构从数组和对象提取值并赋值给独特的变量
[]表示被解构的数组, x,y,z表示要将数组中的值存储在其中的变量,
在解构数组是, 还可以忽略值, 例如const[x,,z]=point,忽略y坐标

const point = [1,2,3];
const [x,y,z] = point;
console.log(x,y,z)  //1 2 3

结构对象中的值
花括号 { } 表示被解构的对象，type、color 和 karat表示要将对象中的属性存储到其中的变量

const point = {
    name:'迪丽热巴',
    age:'22',
    text:'还欠我一本结婚证'
}
const {name,age,text} = point;
console.log(name+'今年'+age+'岁'+text)  //迪丽热巴今年22岁还欠我一张结婚证

对象字面量简写

const a = '苹果';
const b = '香蕉';
const c = '菠萝';
const obj = {a,b,c};
console.log(obj) //{a: "苹果", b: "香蕉", c: "菠萝"}
